# POWER PLANT - Energy Production Prediction

This project predicts energy output using 4 input features:
   -Temperature
   -Humidity
   -Vaccum
   -Pressure
It is a regression problem solved using multiple machine learning models and ANN.

# Models Used
-Linear Regression
-KNN Rgressor
-Decision Tree Rgressor
-Random Forest Regressor
-Gradient Boosting Rgressor
-Voting Regressor
-Artifial Neural Network 

# Approach
-Data preprocessing(scaling data)
-Train-test split
-Model training
-Visualization
-Evaluation using R² Score

# Best Model -->> Gradient Boosting Regressor with R² Score: 96.34%
